# UNIQUE Registry

The first permanent on-chain ticker name standard on BNB Chain.

One ticker. One token. Forever.

## Contract

**BNB Smart Chain (Mainnet)**
`0x12F5D57243b8F8b7897ABDC8530950dd1071E74e`

[View on BscScan](https://bscscan.com/address/0x12F5D57243b8F8b7897ABDC8530950dd1071E74e)

## What it does

UNIQUE Registry permanently associates a ticker with a smart contract address on-chain. Once registered, no other token can claim the same ticker on integrated platforms. No duplicates. No clones.

## Features

- Ticker registration (1 to 20 characters, full Unicode support)
- Pre-launch reservation up to 7 days before deployment
- Batch registration up to 50 tickers in a single transaction
- Launchpad certification badge for integrated platforms

## Security

- ReentrancyGuard on all payable functions
- 48-hour TimeLock on all fee and ownership changes
- Source code verified and open-source on BscScan (MIT License)
- Third-party audit planned
